Задание 1.
Дан код:

var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3

Почему код даёт именно такие результаты?

Решение.
1. c = ++a; alert(c);           // 2
Значение переменной a = 1, используется префиксная операция добавления 1 к значению переменной а, соответственно сначала добавляется 1 к значению переменной a (1), 
потом результат (2) записывается в переменну с и выводиться на экран.
2. d = b++; alert(d);           // 1
Значение переменной b = 1, используется постфиксная операция добавления 1 к значению переменной b, соответственно сначала значение переменной b (1)
присваивается переменной d, а только потом к значению переменной b (1) добавляется 1. Значение переменной d = 1, b = 2. На экран выводится значение переменной d
3. c = (2+ ++a); alert(c);      // 5
Значение переменной a = 2 (результат выполнения пункта 1), используется префиксная операция добавления 1 к значению переменной a, соответственно сначала добавляется 1 
к значению переменной a (1), затем происходит сложение 2 + 3, результат (5) записывается в переменную с и выводится на экран.
4. d = (2+ b++); alert(d);      // 4
Значение переменной b = 2 (результат выполнения пункта 2), используется постфиксная операция добавления 1 к значению переменной b, соответственно сначала значение 
переменной b (2) складывается с числом 2, результат (4) записывается в переменную d, потом к значению переменной b прибавляется 1. Значение переменной d = 4, b = 3. 
На экран выводиться переменная d.
5. alert(a);                    // 3
Значение переменной a = 3 (результат выполнения пункта 3).
6. alert(b);                    // 3
Значение переменной b = 3 (результат выполнения пункта 4).
